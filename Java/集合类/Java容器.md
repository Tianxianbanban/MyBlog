## Java容器



Java容器主要分为Collection 和 Map，Collection 存储着对象的集合，而 Map 存储着键值对（两个对象）的映射表。

**Collection**

+ Set
  + TreeSet
  + HashSet
  + LinkedHashSet
+ List
  + ArrayList
  + Vector
  + LinkedList
+ Queue
  + Deque
    + LinkedList
    + ArrayDeque
  + PriorityQueue

**Map**

+ HashTable
+ HashMap
+ TreeMap
+ LinkedMap



**并发容器**

+ ConcurrentHashMap
+ CopyOnWriteArrayList
+ LinkedBlockingQueue
+ ArrayBlockingQueue
+ DelayQueue
+ SynchronousQueue
